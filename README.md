# TechHive Web Application
An online store for cutting-edge **electronics** and **smart home devices**. 

## üé• Demo
Check out the live demo: [Live Demo](https://techhive-ecommerce-platform.onrender.com/) 

## üöÄ Features
### User Features
- Browse & Search Products with filtering
- View Product Details & Reviews
- Add to Cart & Favourites
- Place Order & Finalise Payment
- Track Orders
- User Authentication
### Admin Features
- Add, edit, and delete categories and products
- View and manage registered users
- Track and process customer orders
- Gain insights into sales trends and performance

## üèóÔ∏è Tech Stack
TechHive is built using the **MERN Stack**
- **MongoDB** ‚Äì NoSQL database for storing product, user, and order data.
- **Express.js** ‚Äì Backend framework for handling API requests.
- **React.js** ‚Äì Frontend library for an interactive user interface.
- **Node.js** ‚Äì Runtime environment for backend operations.
